# Roadmap

## Pass 1 — MVP (current)
Flask skeleton, lesson/exercise pages, notes, progress tracking, 3 lessons (pwd, ls, cd).

## Pass 2 — Interactive Terminal
Fake JS terminal on exercise pages. Command input, validation, step-by-step feedback.

## Pass 3 — Expand Content
More commands: `cat`, `grep`, `find`, `chmod`, `man`. Second module: File Inspection.

## Pass 4 — Persistence
SQLite for progress. Multi-device sync. Possibly light user accounts.

## Later
Full curriculum: networking basics, reconnaissance tools, web fundamentals.
